- 每个工具目录同时提供 config.json（功能/协议/命令模板/配置映射）与 paths.json（平台可执行探测路径），职责分离便于跨平台扩展。
- config.json 中的 write 映射采用「目标路径 → 源字段」键值对形式，将 AnyVersion 内部字段（model/baseUrl/apiKey/fallbackModel）扁平化写入第三方配置文件。
- 工具通过 supportsOpenai/supportsAnthropic/supportsGoogle 三布尔位声明兼容的 API 协议族，再由 providers.json 中的 openaiUrl/anthropicUrl/googleUrl 提供具体端点，实现协议与地址解耦。
- session 存储使用 type + dirs 组合描述（claude_projects/jsonl/opencode_style），dirs 支持多路径以便兼容不同发行版的默认位置。
- MCP 与 Skills 扫描采用「中心索引 + 工具枚举」模式：mcp-config.json 与 skills-scan.json 仅做注册，不内嵌具体配置内容。
