- 对外暴露的公共 API 一律使用 `#[tauri::command]` 宏标注函数，返回 `Result<T, String>` 错误形式，由 Tauri 序列化后传给前端。
- 对 Windows 注册表的环境变量写入统一走 `set_registry_env` / `set_system_registry_env`，并在写入后调用 `broadcast_setting_change()` 发送 `WM_SETTINGCHANGE` 通知系统刷新。
- 长耗时 I/O 操作（目录复制、迁移、清理）通过 `app_handle.emit("xxx-progress", ProgressStruct)` 向 UI 推送阶段/进度/文件名，Progress 结构体统一包含 `stage/current/total/file_name` 字段。
- 路径常量与默认值集中放在 `config.rs` 的 `get_base_dir()` / `load_config()`，其他模块通过 `super::config::load_config()` 获取，避免硬编码 `~/.any-version`。
- SDK 安装来源识别采用“规则列表 + 优先级排序”模式：在 `FindRule` 中声明 `ResolvePattern`、`priority`、`root_offset`，由 `find_sdk_root` 按序匹配并去重。
- 所有外部命令执行统一经过 `super::hidden_cmd::hidden_cmd(...)` 包装，以隐藏控制台窗口并统一处理 stdout/stderr 输出。
