- Tauri 命令统一放在 `src-tauri/src/commands/<domain>/` 子模块中，并通过 `lib.rs` 的 `invoke_handler![]` 集中注册，不分散在各处。
- 需要修改托盘 UI 的命令（如切换版本、启停服务）在执行后主动调用 `tray::refresh_tray_menu` 或 `rebuild_tray_menu` 刷新菜单，保持视图与状态一致。
- 托盘菜单项 ID 采用前缀常量（`ID_SWITCH_PREFIX`、`ID_SERVICE_START_PREFIX`、`ID_SERVICE_STOP_PREFIX`）配合 `starts_with`/`strip_prefix` 模式匹配路由，而非为每个动作定义独立常量。
- 对可能失败的窗口操作（show/unminimize/set_focus/set_decorations）一律使用 `let _ = ...` 忽略错误，保证托盘交互不被 UI 细节阻塞。
