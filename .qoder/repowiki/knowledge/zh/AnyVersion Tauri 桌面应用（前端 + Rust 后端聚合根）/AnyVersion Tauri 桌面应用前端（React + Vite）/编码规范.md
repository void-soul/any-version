- 页面/Tab 级懒挂载：用 `Set<TabKey>` 记录已访问过的键，仅在 `mountedTabs.has(key)` 为真时才渲染对应子树，默认只预挂载首项。
- Tauri 命令调用统一走 `import { invoke } from "@tauri-apps/api/core"`，并以泛型参数标注返回值类型（如 `invoke<ProjectStatus[]>(cmd)`）。
- Tab 配置集中声明为 `TABS` 数组（key/label/icon 三元组），再由 `.map` 生成左侧导航按钮，避免硬编码分支。
- 侧边栏与内容区采用 `flex` 布局，左侧固定宽度（`w-25`/`w-40`）+ `border-r border-white/5` 分隔，右侧 `flex-1 min-h-0 overflow-y-auto` 自适应滚动。
- 深色主题通过统一的 Tailwind 调色板（`bg-[#0e1220]/80`、`text-slate-400`、`hover:bg-white/5`）和 `select-none` 禁用文本选择，保持桌面应用质感。
