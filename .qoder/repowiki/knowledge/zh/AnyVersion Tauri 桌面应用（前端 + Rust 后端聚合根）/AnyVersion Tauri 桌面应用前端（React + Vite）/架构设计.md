- 入口 `src/main.tsx` 通过 `ReactDOM.createRoot` 挂载 `<App />`；`App.tsx` 作为顶层容器，使用 `data-tauri-drag-region` 实现自定义无边框窗口拖拽，并通过 `@tauri-apps/api/window` 调用 hide/minimize/maximize/close 等原生窗口控制。
- 页面级路由由 `PageId` 联合类型驱动，配合 `mountedPages: Set<PageId>` 实现**懒挂载**：仅首次访问时渲染对应子组件，避免启动时全部初始化。
- 五个主页面按职责拆分到独立组件：`ProjectManager`（SDK 管理）、`AiPanel`（AI 工作区）、`RssReader`（资讯）、`SystemTools`（系统工具集）、`GlobalSettings`（全局设置），每个大模块内部再按 tab 拆分子组件（如 `ai/ModelConfig`、`ai/ToolLauncher` 等）。
- 状态与后端交互统一通过 `@tauri-apps/api/core` 的 `invoke` 调用 Rust 命令（如 `project_list`、`project_status`、`get_config`），前端不直接读写文件系统。
- 组件组织遵循“聚合器 + 展示”模式：`ProjectManager` 聚合列表/详情两侧布局并负责增量更新，`SystemTools` / `AiPanel` 以左侧竖菜单 + 右侧内容区的固定模板复用 tab 切换逻辑。
