        </figure><h3 id="the-mechanics-under-the-hood">The Mechanics Under the Hood</h3><p>An MCP server is like a menu of tools, where each tool is carefully described so AI can understand how to use it. For example, when you look at a Notion MCP server, you'll find tools like <code>create_database</code> or <code>query_database</code>. Each tool includes:</p><ul><li><strong>A Name:</strong> What it's called.</li><li><strong>A Description in Natural Language:</strong> What it does.</li><li><strong>A Schema:</strong> Defines exactly what information it needs.</li><li><strong>The Actual Code:</strong> That makes the API calls.</li></ul><figure class="kg-card kg-image-card"><img src="https://storage.ghost.io/c/d6/fe/d6feb101-a8e6-444b-bae8-3ca714794abb/content/images/2025/02/image-25.png" class="kg-image" alt="" loading="lazy" width="1964" height="1024" srcset="https://storage.ghost.io/c/d6/fe/d6feb101-a8e6-444b-bae8-3ca714794abb/content/images/size/w600/2025/02/image-25.png 600w, https://storage.ghost.io/c/d6/fe/d6feb101-a8e6-444b-bae8-3ca714794abb/content/images/size/w1000/2025/02/image-25.png 1000w, https://storage.ghost.io/c/d6/fe/d6feb101-a8e6-444b-bae8-3ca714794abb/content/images/size/w1600/2025/02/image-25.png 1600w, https://storage.ghost.io/c/d6/fe/d6feb101-a8e6-444b-bae8-3ca714794abb/content/images/2025/02/image-25.png 1964w" sizes="(min-width: 720px) 720px"></figure><p>When Cline wants to create a database in Notion, here's the process:</p><ol><li><strong>Discovery:</strong><br>The MCP server tells Cline what tools are available.</li><li><strong>Selection:</strong><br>Cline sees a tool like <code>create_database</code> along with its description.</li><li><strong>Specification:</strong><br>The server tells Cline exactly what information it needs (for example, where to create it, what to name it).</li><li><strong>Execution:</strong><br>Cline provides that information in the right format, and the server takes care of making the actual API calls to Notion.</li></ol><blockquote><strong>Note:</strong> You still need API keys and proper security. MCP isn't magic; what makes it special is that it creates a standard way for AI to discover and use tools without having to learn the specifics of every API.</blockquote><h2 id="a-mind-blowing-example-of-mcp-in-action">A Mind-Blowing Example of MCP in Action</h2><p>Two months ago, when we first added MCP support to Cline, I witnessed something that completely changed my understanding of what's possible with AI tools.
